# Strong Password Generator (Python)

A Python program to generate strong and secure passwords. The program ensures that each password contains:

- At least one lowercase letter
- At least one uppercase letter
- At least one number
- At least one symbol

It also allows the user to choose the desired password length.

---

## Features

- Guarantees a mix of letters, numbers, and symbols
- Randomized character order for maximum security
- Minimum length check (passwords must be at least 4 characters)
- Easy to extend for additional customization
