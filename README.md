# Strong Password Generator (Python)

A Python-based **strong password generator** that produces secure and unpredictable passwords. Each password contains a mix of:

- At least one **lowercase letter**  
- At least one **uppercase letter**  
- At least one **number**  
- At least one **symbol**  

Users can choose the desired password length, and the generator ensures strong, randomized passwords suitable for real-world use.

---

## Features

- Guarantees a mix of **letters, numbers, and symbols**  
- Randomized character order for **maximum security**  
- Minimum length enforcement for safety (4 or more characters)  
- Modular design with functions for clarity and reusability  
- Easy to extend for additional **customization options**

---

## How It Works

1. **Character Groups:** Separate lists for lowercase, uppercase, numbers, and symbols.  
2. **Guaranteed Characters:** Picks one character from each group to ensure password strength.  
3. **Random Filling:** Fills remaining characters from all available groups.  
4. **Shuffling:** Shuffles the password to make it unpredictable.  
5. **Conversion:** Combines characters into a string for the final password.
